## ⚙️ Setup & Installation
1. Clone the repo:
```bash
git clone <your-repo-url>
cd tutorials-ninja
```
2. Verify Java & Maven installation:
```bash
java -version
mvn -version
```
3. Import project into Eclipse/IntelliJ.  
4. Update `testdata` and `config` values.  
5. Run tests:
```bash
mvn clean test
```
